# HangmanGame
The hangman project game consists of guessing a secret word of a given length by entering one letter at a time. The game ends if the user does not correctly guess the word within the allotted number of guesses. The Hangman game will utilize classes to store the hidden word and guesses a class containing member methods used to track progress, and a class to represent the hangman figure. 

Hangman Made with Python:

A)Features:

1.Visual hangman drawing that updates with each wrong guess

2.5 word categories: Animals, Fruits, Countries, Sports, and Colors

3.Random word selection from the chosen category

4.Tracks correct and wrong letter guesses

5.Win/lose conditions with 6 allowed wrong guesses

6.Clean text-based interface with screen clearing

B)How to play:

--Run the Python script--

1..A random word from a random category will be selected

2..Guess letters one at a time

3.You have 6 wrong guesses before the hangman is complete

4.Win by guessing all letters in the word before running out of tries

5.Type 'quit' at any time to exit

C)Game features:

1.Displays the category as a hint

2.Shows which letters you've guessed (correct and wrong)

3.Updates the hangman drawing with each wrong guess

4.Prevents duplicate letter guesses

5.Option to play multiple rounds

Categories included:

Animals: elephant, giraffe, penguin, dolphin, etc.

Fruits: apple, banana, strawberry, etc.

Countries: france, brazil, japan, etc.

Sports: football, basketball, tennis, etc.

Colors: purple, yellow, crimson, etc.

You can easily add more words or categories by editing the word_categories dictionary. Good luck and have fun playing!
